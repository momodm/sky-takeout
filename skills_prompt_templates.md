# Skills 常用指令模板

## 使用说明

- 下面 10 句都可以直接复制后改少量项目名或需求
- 重启 Codex 后，新安装的 skills 才能在会话里真正被调用
- 如果你不确定用哪个，就先用第 1 句

## 最常用 10 句

1. `帮我从已安装 skills 里选最适合这个项目的，并按对应 skill 开始做`
2. `这个页面改版请使用 frontend-design 和 web-design-guidelines 来做`
3. `这个 React 项目请按 vercel-react-best-practices 优化结构、性能和可维护性`
4. `这个 Next.js 网站请结合 frontend-design 和 vercel-react-best-practices 一起实现`
5. `这个后台管理页面请用 frontend-design 提升视觉质量，再用 web-design-guidelines 调整信息层级`
6. `这个 Remotion 视频项目请按 remotion-best-practices 来设计组件、字幕和时间轴`
7. `这个 Azure OpenAI 项目请使用 azure-ai、azure-storage、azure-deploy 来完成方案和落地`
8. `这个 Azure 项目的身份认证部分请使用 entra-app-registration 和 azure-rbac 来设计`
9. `这个 Azure 服务报错了，请用 azure-diagnostics 和 appinsights-instrumentation 帮我排查`
10. `帮我用 azure-compliance、azure-cost-optimization、azure-resource-visualizer 评估这套 Azure 架构`

## 按项目类型快速套用

### 官网 / 营销页
- `这个官网请使用 frontend-design 和 web-design-guidelines 来完整重做`

### React / Next.js Web 项目
- `这个 Web 项目请使用 vercel-react-best-practices、frontend-design 一起推进`

### 后台管理系统
- `这个后台请用 frontend-design 优化视觉，并按 web-design-guidelines 调整页面结构`

### Remotion 视频项目
- `这个视频生成项目请按 remotion-best-practices 来实现`

### Azure AI 应用
- `这个 Azure AI 应用请使用 azure-ai、azure-storage、azure-deploy`

### Azure 企业治理
- `这套 Azure 资源请按 azure-compliance、azure-rbac、azure-cost-optimization 检查一遍`

## 万能兜底句

- `这个需求你先帮我判断该用哪些已安装 skills，再按最合适的方式开始做`
